
## Descripción

Este proyecto es una página web básica sobre el mundo de los videojuegos. Presenta información sobre empresas importantes y algunos juegos destacados.

## Tecnologías usadas

* HTML5
* CSS3

## Estructura del proyecto

```
/proyecto
 ├── index.html
 └── /assets
      ├── /css
      │    └── style.css
      └── /img
```

## Cómo usar

1. Descargar o clonar el proyecto
2. Abrir el archivo `index.html` en el navegador

## Características

* Menú de navegación
* Secciones organizadas
* Imágenes ilustrativas
* Estilos en CSS
